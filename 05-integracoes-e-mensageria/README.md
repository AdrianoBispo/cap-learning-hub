# Módulo 05: Integrações e Mensageria

![Infográfico sobre Conexão de Serviços](../Infograficos/05.01.01_Dominando_a_Conexão_de_Serviços_com_CAP.png)

Aplicações modernas raramente vivem isoladas. Este módulo aborda as diferentes formas de integrar sua aplicação CAP com outros sistemas: consumindo serviços remotos (síncronos) e implementando padrões de mensageria assíncrona para construir arquiteturas mais resilientes e desacopladas.

## Tópicos

| Tópico | Descrição |
| :--- | :--- |
| [**Consumo de Serviços Remotos**](./01-servicos-remotos/README.md) | Como integrar e consumir serviços externos (OData, REST) dentro de uma aplicação CAP, tratando-os como se fossem locais. |
| [**Mensageria Assíncrona e Eventos**](./02-mensageria-assincrona/README.md) | Explora a arquitetura orientada a eventos do CAP, mostrando como emitir e consumir eventos para construir microserviços desacoplados. |
| [**Infográficos do Módulo**](./Infograficos/README.md) | Coleção de todos os infográficos e diagramas visuais relacionados a este módulo. |

---

Aprender a integrar efetivamente sua aplicação CAP é essencial para construir soluções que se conectam ao ecossistema SAP e além, aproveitando a modularidade e a escalabilidade.
