# Módulo 03 (Parte 2): Drivers e Serviços de Banco de Dados

![Infográfico sobre o DatabaseService](../Infograficos/03.02.02_DatabaseService_Masterclass.png)

Este módulo foca na camada de serviço de persistência do CAP, o `DatabaseService`. Ele aborda como o CAP se conecta e interage com o banco de dados (por exemplo, SAP HANA) e como é possível estender o comportamento padrão para implementar lógicas customizadas.
