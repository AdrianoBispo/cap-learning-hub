# Módulo 03 (Parte 1): Consultas com CQL e CQN

![Infográfico sobre CQL](../Infograficos/03.01.01_CQL_Evolucao_SQL.png)

Este módulo explora a forma como o CAP abstrai o acesso a dados através da **CDS Query Language (CQL)** e da sua representação em objeto, a **CDS Query Notation (CQN)**. Dominar CQL e CQN é essencial para interagir com a camada de persistência de forma eficiente e segura.
