# Módulo 02: Runtime do Node.js e Lógica de Negócios

![Infográfico sobre o Runtime do Node.js](../Infograficos/02.01.01_Nodejs_Server_Mastery.png)

Este módulo mergulha no coração do SAP CAP com Node.js. Ele cobre como o servidor é inicializado, como os serviços são definidos e implementados, e como a lógica de negócios customizada é adicionada através de event handlers. Ferramentas essenciais do SDK e a configuração com TypeScript também são exploradas.

| [**Event Handlers e Serviços**](./02-event-handlers-e-servicos/README.md) | Detalha como registrar handlers para eventos (CRUD), a dualidade dos serviços (API vs. implementação) e o fluxo de requisições. |
| [**Ferramentas do SDK e Utilitários**](./03-ferramentas-sdk-utils/README.md) | Apresenta as ferramentas do SDK do CAP que simplificam tarefas comuns, como o gerenciamento de transações. |
| [**Setup com TypeScript**](./04-typescript-setup/README.md) | Orienta como configurar e utilizar TypeScript em projetos SAP CAP para obter um desenvolvimento mais robusto e seguro. |
