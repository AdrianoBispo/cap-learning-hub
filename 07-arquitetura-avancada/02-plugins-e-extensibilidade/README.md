# Módulo 07 (Parte 2): Plugins e Extensibilidade

![Infográfico sobre Extensibilidade em CAP](../Infograficos/07.02.01_Extensibility_Toolkit.png)

A extensibilidade é crucial para aplicações que precisam se adaptar a requisitos de negócio em constante mudança. Este módulo aborda a arquitetura de extensibilidade do CAP, permitindo que as aplicações sejam customizadas e estendidas sem modificar o código-fonte original, utilizando plugins e extensões modulares.
