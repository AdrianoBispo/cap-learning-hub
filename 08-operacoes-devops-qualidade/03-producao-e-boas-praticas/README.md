# Módulo 08 (Parte 3): Produção e Boas Práticas

![Infográfico sobre do `cds watch` à Produção](../Infograficos/08.03.01_Do_cds_watch_a_Producao.png)

A transição de uma aplicação do ambiente de desenvolvimento para produção exige planejamento e a adoção de boas práticas. Este módulo serve como um guia para levar suas aplicações CAP de desenvolvimento à produção, cobrindo tópicos cruciais como otimização de performance e as melhores práticas de desenvolvimento em Node.js.
