# Módulo 01 (Parte 4): Modelagem Performática e Aspectos

![Infográfico sobre Modelagem Performática](../Infograficos/01.04.01_Performance_Modeling.png)

Este módulo foca em técnicas avançadas de modelagem em CDS para otimizar a performance e reutilizar definições através de aspectos. Aprender a modelar com performance em mente e a usar aspectos de forma eficaz pode acelerar o desenvolvimento e melhorar a manutenibilidade do seu código.
