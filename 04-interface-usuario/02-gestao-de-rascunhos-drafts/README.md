# Módulo 04 (Parte 2): Gestão de Rascunhos (Drafts)

![Infográfico sobre Gestão de Rascunhos](../Infograficos/04.02.01_Nodejs_Rascunhos_Guia.png)

Este módulo detalha o suporte a "drafts" (rascunhos) em CAP. Esse é um recurso poderoso que melhora a experiência do usuário, permitindo que alterações em objetos de negócio sejam salvas automaticamente em um estado de rascunho antes de serem persistidas definitivamente na tabela principal.
