# Módulo 04: Interface de Usuário com Fiori Elements

![Infográfico sobre Fiori Elements](../Infograficos/04.01.01_Fiori_Elements_Metadados.png)

Uma das grandes vantagens do SAP CAP é sua capacidade de gerar UIs de alta qualidade com baixo esforço. Este módulo aborda como usar anotações de metadados para renderizar aplicações **SAP Fiori Elements** e como habilitar o recurso de "drafts" para melhorar a experiência do usuário em cenários de edição de dados.

## Tópicos

| Tópico | Descrição |
| :--- | :--- |
| [**Fiori Elements e Anotações**](./01-fiori-elements-annotations/README.md) | Mostra como o CAP gera UIs ricas através de anotações (`@UI.*`) nos modelos de serviço, traduzindo a intenção do modelo em uma interface funcional. |
| [**Gestão de Rascunhos (Drafts)**](./02-gestao-de-rascunhos-drafts/README.md) | Detalha o suporte a "drafts", que salva alterações automaticamente, melhorando a robustez e a experiência de uso da aplicação. |
| [**Infográficos do Módulo**](./Infograficos/README.md) | Coleção de todos os infográficos e diagramas visuais relacionados a este módulo. |

---

Ao final deste módulo, você será capaz de construir o backend para aplicações Fiori Elements completas, declarando a estrutura e o comportamento da UI diretamente no seu modelo de serviço.
